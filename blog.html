What is Document Object Module?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree of nodes. Each node represents a part of the document, such as an element, attribute, text node, or comment. The DOM provides a way to access and manipulate these nodes using JavaScript

How do you select an element from the DOM?

There are several ways to select an element from the DOM using JavaScript. We can select element by using id, class name, query selector and tag name.

What is the event delegation in the context of DOM, and why it is useful?

Event Delegation in the context of the DOM is a technique where you attach an event listener to a parent element instead of directly to each child element. This allows you to handle events for multiple elements efficiently, especially when the number of child elements is dynamic or large.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attribute first select the element, then get the attribute and finally set the value of the attribute.
to manipulate the style of an element, first select the element, then add css using element.style.cssProperty = "value".
